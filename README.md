# 프로젝트 선정 이유
#### Youtube 혹은 OTT 같은 서비스에서 추리게임을 다루는 프로그램을 시청한 적이 있었습니다. 이때, 한 카드게임의 진행방식이 굉장히 흥미로웠습니다. 그렇게 느낀 이유는 바로 카드 한장을 본인의 이마 앞으로 가져간다는 점 때문이었습니다. 이러한 경우라면,  본인은 볼 수 없지만, 상대방은 볼 수 있었기 때문에 심리를 이용하여 게임을 진행하여야 한다는 점이 인상적이었습니다. 이뿐만 아니라, 게임의 규칙들을 코드로 작성하여 완성시킨다고 생각하니 고려해줄 것이 많다고 느껴졌고 더욱더 관심을 가지게 되었습니다. 해당 카드게임의 명칭을 정확히 알고 싶어서 구글링을 하던 도중, 이 게임이 '인디언 블랙잭'임을 알게 되었고 프로젝트로 선정하게 되었습니다.
